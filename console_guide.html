Minecraft コンソール版 - サーバーの入り方

コンソール版サーバーに接続する方法

  1. **更に詳しい接続方法はこちらで紹介しています。
  2. **Microsoftアカウントにログイン**
    Minecraftにログインするには、Microsoftアカウントが必要です。
  3. **svcn1 または svcn2 をフレンドとして追加**
    マイクラの「フレンド追加」で「svcn1」または「svcn2」と入力し、フレンドリクエストを送信します。
  4. 入れない場合こちらのサイトを御覧くださいhttps://www.svcn.pgw.jp/gemertag
  5. **外部サーバーへ接続**
    フレンドの欄に戻り、「外部サーバーへ接続」を選択します。
  6. **Manage Server Listを選択**
    「Manage Server List」をクリックし、「Add a Server」を選択します。
  7. **サーバー情報を入力**
    以下の情報を入力してください：
    • Server Address: single-ocean.gl.at.ply.gg
    • Server Port: 3681
    • Display Name on Server List: お好きなサーバー名を入力（任意）
  8. **送信をクリック**
    「送信」をクリックして設定を保存します。
  9. **サーバーに接続**
    最初のページに戻り、追加したサーバーを選択してゲームを開始します。

さらにサポートが必要な場合は、こちらの公式Discordをご覧ください。
